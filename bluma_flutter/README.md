# Cicla

A Period tracking app built with Flutter.
This is a Flutter port of [Bluma](https://github.com/blumaHQ/bluma-app), originally developed by Maribel Ferreira.
This project is an independent Flutter rewrite and is not affiliated with or endorsed by the original Bluma project.

## Getting Started

```bash
git clone https://github.com/yourusername/cicla.git
cd cicla
flutter pub get
flutter run 
```

## Disclaimer
Cicla is still in trial and not production ready. Please always consult a healthcare professional for medical advice.

## Privacy
All data is stored locally on the device. Cicla does not collect or share personal data.
